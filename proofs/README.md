# Proofs

Committed evidence that each feature actually works. One subfolder per
feature/milestone, each with a `PROOF.md` describing **what is shown** and **how to
reproduce it**, alongside the artifacts (captured requests, tool/schema dumps,
before/after state, conformance reports, and — for the v2 monitor — Playwright
screenshots).

See [`../CLAUDE.md`](../CLAUDE.md) §6 for the proof discipline and
[`../specs/DESIGN.md`](../specs/DESIGN.md) §20 for the per-feature proof methods.

Throwaway tooling used to produce a proof belongs in the gitignored `scratchpad/`,
not here.
